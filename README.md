# MindData – Dashboard de Análise IA

Este repositório contém a prova de conceito da Inteligência Artificial da app MindData, desenvolvida para o concurso NOVA IMS CHALLENGE – APPLICA-TE 2026.

## Funcionalidades
- Gráfico de linhas (evolução humor e sono)
- Gráfico de barras (ecrã vs atividade física)
- Gráfico de radar (comparação com metas ideais)
- Avaliação de risco (ansiedade, depressão, isolamento)
- Recomendações personalizadas

## Como executar
1. `npm install`
2. `npm start`
